# BriskORM

BriskORM is a fast, light-weight, brisk ORM to work in nodejs,support mongodb. Use the brisk-ioc as the IoC/DI container.

BriskORM 是一个快速，轻量级，轻快的ORM工作在nodejs，支持mongodb。使用brisk-ioc作为IoC/DI容器。
